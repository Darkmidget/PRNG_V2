# Solution: Make sure Vivado is installed and accessible

# Solution: Close other terminal programs (PuTTY, serial monitor, etc.)

# Issue: "No data received after enabling SW[0]"
# Solution:
# - Verify LED[1] (heartbeat) is blinking
# - Try pressing SW[0] button multiple times
# - Check USB cable connection
# - Verify COM4 with: [System.IO.Ports.SerialPort]::GetPortNames()

## Modified Verilog Changes (ring_osc.v)

# KEY MODIFICATION:
# Original: Sends 1 sample/sec (update_tick = 1 Hz)
# New: Sends continuously as soon as previous transmission completes
#
# Location: Line ~250 in ring_osc.v
#
# Changed from: if (update_tick && !tx_busy) begin
# Changed to:   if (!tx_busy) begin
#
# Effect: LFSR value sampled every ~52 microseconds at 115200 baud
# Result: ~115,000 samples possible per second
#         100,000 samples in ~0.87 seconds
